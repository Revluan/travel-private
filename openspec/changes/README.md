# Active changes

Every subdirectory here (except `archive/`) is an in-progress OpenSpec change.

Each change folder has:

- `proposal.md` — why and what
- `specs/` — requirements and scenarios (the delta being proposed)
- `design.md` — how
- `tasks.md` — the checklist

Use `/status` to see a summary, or read any folder directly.

When a change is done, `/archive <name>` moves it to `archive/<YYYY-MM-DD>-<name>/` and merges its specs into `openspec/specs/`.
